# Матрица компетенций для студентов, изучающих программирование

Данный репозиторий содержит структурированную матрицу компетенций (набор знаний и навыков) для студентов, осваивающих разработку на C#, .NET, SQL, веб-технологии и смежные области. Матрица помогает систематизировать обучение, проводить самооценку и отслеживать прогресс.

## О проекте

Матрица компетенций разработана на основе анализа профессиональных требований и лучших практик. Она охватывает следующие области:

- **Computer Science** — структуры данных, алгоритмы, архитектура и паттерны.
- **Язык C# и .NET** — от основ до продвинутых тем, включая GUI, доступ к данным, тестирование, ASP.NET Core и документирование.
- **Базы данных: SQL и СУБД** — основы SQL, особенности PostgreSQL, MySQL, SQLite.
- **Веб-разработка** — HTML/CSS, JavaScript, PHP.
- **Мобильная разработка** — Android с Kotlin и Jetpack Compose.
- **Контейнеризация** — Docker.
- **Инструменты** — Git.
- **Soft skills** — взаимодействие в команде.
- **Профессиональные практики** — отладка, профилирование, проактивность.

Каждый навык классифицирован как **hard** (технический) или **soft** (универсальный). Это позволяет оценивать как техническую подготовку, так и личностные качества.

## Форматы данных

Матрица представлена в четырёх форматах (все хранятся в папке `data/`):

- **JSON** (`competence_matrix.json`) — для веб-приложений, мобильных приложений, API.
- **XML** (`competence_matrix.xml`) — для систем, требующих строгой схемы (например, enterprise-решения).
- **Markdown** (`competence_matrix.md`) — для чтения человеком и документации.
- **CSV** (`competence_matrix.csv`) — для работы в Excel / Google Таблицах (самооценка).

Все форматы синхронизированы и соответствуют единой схеме (JSON Schema и XSD прилагаются).

## Генератор статического сайта

Проект включает **генератор статического HTML-сайта** на Node.js, который преобразует JSON-данные в красиво оформленную веб-страницу с цветовой дифференциацией категорий и типов навыков.

### Возможности

- Загрузка данных из `data/competence_matrix.json`
- Валидация JSON по схеме (`data/competence_matrix.schema.json`)
- Компиляция LESS в CSS
- Рендеринг Pug-шаблона
- Генерация итоговой страницы в папку `dist/`

### Использование

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите сборку:
   ```bash
   npm run build
   ```
Готовый сайт появится в папке `dist/`. Откройте `dist/index.html`.

#### Скрипты для разработки:

- `npm run lint:js` – проверить JavaScript-код
- `npm run lint:css` – проверить LESS-стили
- `npm run format` – отформатировать код
- `npm run validate` – проверить JSON по схеме
- `npm run build` – полная сборка (валидация + очистка + генерация)

## Методы оценки

В файле [`assessment_methods.md`](assessment_methods.md) подробно описаны:

- Шкалы самооценки (Dreyfus, частота применения, уверенность).
- Методы внешней оценки (тестирование, кейсы, наблюдение, проверка артефактов).
- Комбинированный подход с формулой для итоговой оценки.
- Инструменты для проведения оценки (Google Forms, GitHub Classroom и др.).


## Тестовые задания

Для многих навыков в файле [`test_tasks.md`](test_tasks.md) приведены готовые практические задания с ожидаемым результатом и критериями оценки по 5-балльной шкале. Задания помогут объективно проверить уровень владения навыком.

Пример задания для LINQ:
> Дан список строк: `["apple", "banana", "apricot", "cherry"]`. С помощью LINQ выберите все слова, начинающиеся на 'a', и отсортируйте их по длине.
